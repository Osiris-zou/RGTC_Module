`final/metrics_summary.csv` supplies the aggregate FID, LPIPS, MS-SSIM, runtime, and memory values. `final/paired_stability.csv` supplies paired confidence intervals computed from full-precision per-sample metric values. Per-sample files are stored separately. Absolute workstation paths in the original logs have been converted to portable relative paths.

Note on manuscript rounding: the manuscript table reports the displayed paired differences after rounding the metric means to the shown decimal precision, whereas `paired_stability.csv` retains full-precision raw differences. Therefore, the last decimal digit of very small differences may differ slightly. For example, the raw LPIPS paired improvement is 0.0002084818, while the manuscript displays +0.000209 from the rounded table values. This rounding difference does not affect the interpretation because all reported confidence intervals overlap zero.
